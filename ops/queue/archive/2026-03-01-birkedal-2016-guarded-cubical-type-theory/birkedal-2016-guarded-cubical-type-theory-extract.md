# Extract Task: birkedal-2016-guarded-cubical-type-theory

Birkedal, Bizjak, Clouston, Grathwohl, Spitters, Vezzosi (2016).
"Guarded Cubical Type Theory: Path Equality for Guarded Recursion."
arXiv:1606.05223v2.

Combines guarded dependent type theory (GDTT) with cubical type theory (CTT)
to provide computational interpretation of extensionality for guarded recursive types
while retaining decidable type-checking potential.
